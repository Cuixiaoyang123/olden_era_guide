[喷酸虫](/units/spitter/) 的升级形态。**双发射击 + 酸液腐蚀升级到 -3 防御 / 4 回合** —— 重型喷酸虫是虫巢中后期的**远程核心 + 防御 debuff 主源**。

## 数据解读

| 属性 | 喷酸虫 | 重型喷酸虫 |
|---|---|---|
| 攻防 | 11/9 | 13/10 |
| 伤害 | 7-10 | 8-12 |
| 血量 | 38 | 48 |
| 射击数 | 12 | 16 |
| 双发 | ❌ | **✅** |
| 酸液 -防御 | -2 / 3 回合 | **-3 / 4 回合** |
| 招募 | 380 | 520 |

升级费用 3500 金 + 4 水晶 + 2 宝石。**虫巢玩家第四周末必升**。

## 双发射击的实际输出

重型喷酸虫的**单回合两次酸液弹道**：

- 单次主目标 8-12 × 2 次 = 16-24 主目标伤害
- 路径式弹道 ×2 = 路径上目标各受 4-6 × 2 = 8-12 路径伤害
- **酸液腐蚀双发触发 = 防御 -3 持续 4 回合**

实战：一队 10 重型喷酸虫双发命中：

- 主目标：160-240 伤害
- 路径目标（假设 3 个）：每个 80-120 伤害
- **单波理论总伤害：400-600**

## 酸液腐蚀 -3 / 4 回合

防御减益从 -2 / 3 回合 → **-3 / 4 回合**。意味着：

- 单次命中后**4 回合内**所有友方攻击对该目标 +3 防御穿透
- 配合 [战斗工虫](/users/warrior-drone/) / [蜂后侍卫](/users/royal-guard/) 的近战 → 防御 -3 让物理输出实际放大约 15-20%
- 配合 [蜂巢主宰](/users/hive-sovereign/) 的全军 +2/+2 buff → 净效果攻击 +2 / 防御穿透 +3

虫巢"远程 buff + 近战收割"组合：

1. 重型喷酸虫上酸液腐蚀（防御 -3）
2. 战斗工虫 / 精英战斗工虫冲锋（伤害放大 20%）
3. 蜂后侍卫 / 蜂巢主宰收尾

## 路径式弹道升级

升级后路径式弹道**未变**（依然 50% 路径伤害，不分敌我）。重型喷酸虫**依然会射穿己方阵型**。

但双发射击意味着：

- 一发酸液射出 → 穿过 3-4 个敌方单位 → 第二发再射出 → 穿过另一条路径
- 实战中虫巢玩家可以**让重型喷酸虫主动调整目标** —— 第一发打主目标 + 第二发打另一翼

## 使用要点

**站位**：阵型边缘 —— 让酸液弹道穿过敌方阵型，远离己方密集阵型。

**优先目标**：

1. **敌方密集前排**（3+ 单位排成一线）
2. **敌方高防御核心**（[圣堂骑士](/users/paladin/) / [战独角兽](/users/war-unicorn/) / [那迦](/users/naga/)）—— 酸液腐蚀 -3 防御质变
3. **不要打单兵** —— 路径式弹道在单目标场景下浪费

**对线神殿**：重型喷酸虫的双发 + 酸液腐蚀让虫巢打神殿的持续输出**质变到接近学院** —— 神殿密集阵型 + 酸液腐蚀 + 配合战斗工虫冲锋 = 神殿前排崩盘。

**对线 [林地](/factions/sylvan/)**：林地的远程齐射（[星辉射手](/users/starlight-archer/) / [森林游侠](/users/forest-ranger/)）比重型喷酸虫先动 —— 林地速度 8+ vs 喷酸虫 4。**虫巢对线林地时重型喷酸虫常常被一回合打废**。

**对线 [亡灵](/factions/necropolis/)**：酸液腐蚀对亡灵正常生效，但路径式弹道射穿亡灵阵型时**也会穿过虫巢自己人** —— 双方阵型都密集，喷酸虫在镜像决战中容易误伤。
