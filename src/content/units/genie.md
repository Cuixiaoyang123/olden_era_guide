学院五级兵。**飞行 + 元素灾难 AOE + 元素亲和** —— 神灯魔仆是学院**机动性最强**的中段单位。

## 数据解读

| 属性 | 神灯魔仆 |
|---|---|
| 攻防 | 14/13 |
| 伤害 | 13-16 |
| 血量 | 65 |
| 速度 | 8 |
| 初动 | 11 |
| 招募 | 700 |

数值上是合格的五级飞行单位 —— 速度 8 + 初动 11 让神灯魔仆**先于 [梦魇](/units/nightmare/) / [战斗工虫](/users/warrior-drone/) 行动**。

## 元素灾难（主动 AOE）

每场战斗 1 次，**3×3 范围 60 固定元素伤害**：

- 范围内**所有敌人**受 60 伤害（不区分敌我？—— 不，**仅敌人**，元素灾难是友伤免疫）
- 元素属性 = 火 / 水 / 风 / 土 中随机一种（影响目标的元素抗性）
- **不消耗英雄法力** —— 是单位自带的"免费 AOE"

战术应用：

- 替代 [火球](/spells/fireball/) 的早期 AOE（在学院魔法行会 3 级前就有 3×3 范围伤害）
- 配合英雄火球 → **单回合两次 3×3 AOE** = 9 格 × 60 + 9 格 × 80 = 1260 总伤害
- 对密集敌方阵型杀伤力极大

## 元素亲和（核心被动）

**所有元素类伤害减少 25%**：

- 火球 / 闪电术 / 寒冰射线 / 陨石雨 → 神灯魔仆身上 -25% 伤害
- **黑龙 / 红龙的龙息**也是火系，神灯魔仆吃龙息少 25%
- **巫妖的范围伤害 / 喷酸虫的酸液** = 部分元素属性，也享受减免

这让神灯魔仆是**学院镜像对决**的核心抗法师单位 —— 索兰对线撒迦决战时，神灯魔仆的元素亲和让陨石雨 / 寒冰射线在己方阵列中危险性降低。

## 与 [梦魇](/units/nightmare/) 的对比（五级镜像）

学院 vs 地牢的五级兵对比：

| 单位 | 阵营 | 主特性 | 角色 |
|---|---|---|---|
| 神灯魔仆 | 学院 | 飞行 + AOE + 元素亲和 | 远程 + 控场 |
| [梦魇](/units/nightmare/) | 地牢 | 双格冲锋 + 恐惧光环 + 火焰免疫 | 冲锋 + 控场 |

直接对决：梦魇赢（双格冲锋 + 攻击力优势）。但**神灯魔仆的飞行 + 元素灾难让它在密集阵型对决中能扳回 30-40% 战损**。

## 使用要点

**站位**：飞行单位，可以选择任何位置。学院标准打法 = 神灯魔仆**第一回合飞到敌方密集阵型上空 → 元素灾难 AOE**。

**元素灾难时机**：

- **决战第一回合**（最大化 AOE 覆盖 + 配合英雄火球）
- **敌方密集冲锋阵型**（虫巢集群 / 林地连体阵型）
- **不要打分散阵型** —— AOE 浪费

**优先目标**：

1. 敌方核心远程（高密度堆叠）
2. 敌方治疗系
3. **不要打七级兵单兵堆叠** —— 元素灾难对单目标只是 60 伤害，不如普攻

**对线地牢**：神灯魔仆的元素亲和让 [撒迦](/heroes/shakti/) 的火球 / 闪电术伤害减少 25%。**学院打地牢时神灯魔仆是核心抗法师单位**。

**对线神殿**：神灯魔仆的元素灾难对神殿密集阵型（长矛 + 弓兵 + 圣战士 三层）杀伤力极大。**学院打神殿时神灯魔仆的元素灾难是决战关键**。

**升级判断**：升级版"主神灯魔仆"获得**元素灾难升级到 5×5 范围 + 80 伤害 + 每场可释放 2 次**。**学院玩家第五周末必升**。
