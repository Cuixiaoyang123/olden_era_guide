[德鲁伊](/units/druid/) 的升级形态。**召唤次数翻倍 + 法术池扩展 + 大自然之友被动** —— 林地终局阵容的"无所不能工具人"。

## 数据解读

| 属性 | 德鲁伊 | 大德鲁伊 |
|---|---|---|
| 攻击 | 13 | 15 |
| 血量 | 55 | 65 |
| 射击数 | 16 | 20 |
| 召唤次数 | 1 / 场 | **2 / 场** |
| 召唤树精数量 | 3 | **4** |
| 法术池 | 加速 / 迟缓 | **加速 / 迟缓 / 自然光辉** |
| 招募 | 550 | 700 |

升级费用 4000 金 + 4 水晶 + 2 水银。**林地玩家第五周末必升**。

## 自然光辉（新增法术池）

"自然光辉"是林地专属的轻量级治疗法术：

- 恢复目标友方 50 + 5 × 大德鲁伊数量 HP
- **解除流血 / 中毒 / 凋零**（同 [治疗术](/spells/cure/) 第一档）
- 不消耗英雄法力
- **不解负面 buff**（迟缓 / 诅咒不解）

战术意义：

- 林地的"内置治疗" —— 阿拉里克本身学不到光系，大德鲁伊填补这个空洞
- 配合 [大自然之友](/units/elder-dryad/) 的反伤体系，**林地后期的"打不死"体系成型**

## 召唤树精机制升级

每场 2 次召唤 × 每次 4 株 = **单场战斗最多召唤 8 株临时树精**。

战术应用：

- **决战第一回合召唤 4 株挡敌方冲锋** → 第五回合再召 4 株挡决战收尾
- **召唤位置在敌方核心旁** → 临时树精挡线 + 大自然之友被动加成 → 单回合反伤累积
- 树精分身**血量 35**（比基础召唤 30 略高），能扛 1-2 回合敌方冲锋

## 大自然之友（光环被动）

林地阵营核心协同：

- 当敌方攻击 [树精](/units/dryad/) / [远古树精](/units/elder-dryad/) 触发反伤时
- **大德鲁伊存在的话，反伤额外 +5 固定伤害**

实战：

- 敌方风暴龙链式闪电打 3 株远古树精 → 每只反伤 30% + 大德鲁伊额外 +5 × 3 = 反伤总额 + 15 固定
- 配合 [战独角兽](/users/war-unicorn/) 守护光环 +30% 法抗，林地后期的"反伤 + 治疗 + 法抗"三位一体成立

## 使用要点

**法术选择决策树**：

| 战况 | 选择 |
|---|---|
| 敌方有强势冲锋单位（梦魇 / 圣堂骑士） | 迟缓术 |
| 己方核心远程（星辉射手 / 凤凰）需先动 | 加速术 |
| 友方核心受伤 50% 以上 | 自然光辉 |
| 都没问题 | 优先加速己方七级兵 |

**站位**：后排核心位。林地阵型 = 树精挡线 + 大德鲁伊在二排 + 星辉射手 / 凤凰飞顶。

**对线神殿**：大德鲁伊的自然光辉是林地打神殿持久战的关键 —— [大天使](/units/archangel/) 的圣光打击对林地是 +50% 伤害，大德鲁伊的治疗 + 大自然之友反伤是反制。

**优先目标**：

1. 敌方治疗系（自身远程压制 + 召唤分身分散火力）
2. 敌方七级兵
