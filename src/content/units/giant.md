学院七级兵基础形态。[泰坦](/users/) 的未升级版。**雷电免疫 + 投掷大石** —— 巨人是本作**唯一近远程双修的七级兵**。

## 数据解读

| 属性 | 巨人 | 泰坦 |
|---|---|---|
| 攻防 | 25/25 | 30/30 |
| 伤害 | 40-60 | 45-60 |
| 血量 | 280 | 320 |
| 速度 | 7 | 8 |
| 雷电免疫 | ✅ | ✅ |
| 投掷大石 | 6 发 | **8 发** |
| 招募 | 5500 | 7000 |

巨人是本作**最贵的基础形态七级兵**（5500 金 + 1 水银 + 1 宝石）。升级到泰坦需要 12000 金 + 4 水银 + 2 宝石 + 1 钻石 —— 学院**第六周末才能稳定上线泰坦**。

## 雷电免疫 + 雷电反击（核心被动）

巨人**完全免疫雷电类伤害**：

- [闪电术](/spells/lightning-bolt/) → 0 伤害
- [风暴龙](/units/storm-dragon/) / [暴风之主](/users/storm-lord/) 的链式闪电 → 0 伤害
- **金属克制不触发**（巨人虽然是金属类，但雷电免疫直接覆盖）

**雷电反击**：任何对巨人施放雷电攻击的攻击方，**受 50% 反伤**。

实战意义：

- [风暴龙](/users/storm-dragon/) 链式闪电打巨人 → 巨人 0 伤害 + 风暴龙受 50% 反伤
- 撒迦的闪电术打巨人 → 0 伤害 + 撒迦的英雄魔法被自反 50%（**是否对英雄生效？本作设定细节，主流认为对英雄无反伤**）

## 投掷大石（远程攻击）

每场战斗有 **6 发"投掷大石"**：

- 单次伤害 60-90（介于巨人近战和七级法术之间）
- 投掷大石**不消耗近战回合** —— 巨人远近兼修
- 6 发 = 单场战斗远程总输出 360-540

战术应用：

- 巨人远距离投掷大石 → 等敌人靠近 → 近战 40-60 伤害收尾
- 配合 [神灯魔仆](/users/genie/) 飞过去骚扰 → 巨人远程支援
- **学院唯一"自带远程"的七级兵** —— 战略灵活性远超其他七级

## 与黑龙的对比

| 单位 | 阵营 | 主特性 | 输出 |
|---|---|---|---|
| [黑龙](/users/black-dragon/) | 地牢 | 法术免疫 + 飞行 + 双倍对天使 | 40-50 近战 |
| **巨人** | 学院 | 雷电免疫 + 远程投掷 + 雷电反击 | **40-60 近战 + 60-90 投掷** |

直接对决：取决于巨人能否触发投掷。如果巨人能在黑龙近战前投掷大石 2-3 次，巨人赢；如果黑龙先飞到位近战，黑龙赢（法术免疫让巨人的英雄法术失效，仅靠近战对碰）。

**巨人的优势是开阔地图（远程发挥）**，**黑龙的优势是狭窄地图（直接近战）**。

## 与 [哥布林学徒](/users/gremlin/) 的协同

巨人是工程构造，享受哥布林学徒修复光环：

- 一队 5 巨人总 HP 1400
- 一队 30 哥布林学徒 = 每回合修复 150 HP（每只哥布林学徒 5 HP）
- 持续 5 回合修复 = 750 HP 回血 = 半队巨人复活

学院"巨人 + 哥布林学徒" 是**最持久的七级阵列** —— 配合 [神灯魔仆](/users/genie/) 的元素亲和 + [那迦](/users/naga/) 的反击免疫，学院的工程构造体系几乎无法被消磨。

## 使用要点

**站位**：第二排或远程位 —— 利用投掷大石的距离优势。**巨人速度 7 + 飞行无 = 不要主动冲锋**。

**优先目标**：

1. **敌方七级兵远程**（[森林游侠](/users/forest-ranger/) / [矮精灵](/units/sprite/)）—— 投掷大石单发杀伤
2. **敌方核心远程 / 法师**（高优先级目标）
3. **不要投掷低 HP 集群**（投掷大石对七级兵单兵价值最高）

**对线地牢**：巨人的雷电免疫让风暴龙系完全失效。**学院打地牢时巨人是核心反风暴龙单位**。撒迦的飞龙流面对学院巨人的胜率降低 20%+。

**对线神殿**：巨人吃 [大天使](/users/archangel/) 的圣光打击 +50%。**学院打神殿时巨人优先级降低**，应该用 [神灯魔仆](/users/genie/) + [那迦](/users/naga/) 替代核心位。

**对线 [虫巢](/factions/hive/) / [林地](/factions/sylvan/)**：投掷大石的远程让巨人对密集阵型杀伤力极大 —— 6 发投掷可以打掉一整队 [幼虫](/units/larva/) 集群。

**升级判断**：泰坦是学院的**终极目标** —— 数值全面提升 + 投掷大石升级到 8 发 + **雷电领域**（每回合自动给一个敌方堆叠 30 雷电伤害）。**学院玩家第七周末必升**。
