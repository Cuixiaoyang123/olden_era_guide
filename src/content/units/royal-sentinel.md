[蜂后侍卫](/units/royal-guard/) 的升级形态。**伤害转嫁 50% + 抗物理 25% + 集群护盾扩展** —— 皇家守卫让虫巢的"中央肉墙"从优秀变成**几乎不可击破**。

## 数据解读

| 属性 | 蜂后侍卫 | 皇家守卫 |
|---|---|---|
| 攻防 | 18/20 | 20/22 |
| 伤害 | 14-22 | 16-25 |
| 血量 | 140 | 170 |
| 守护蜂后转嫁 | 30% | **50%** |
| 抗物理 | ❌ | **✅ 25%** |
| 集群护盾覆盖 | 同种 + 幼虫 / 工虫 | **所有虫族单位** |
| 招募 | 1500 | 1900 |

升级费用 5500 金 + 4 宝石 + 2 水银。**虫巢玩家第五周末必升**。

## 守护蜂后 · 加强（核心质变）

伤害转嫁从 30% → **50%**。配合 3 队皇家守卫（虫巢决战标准阵容）：

- 蜂巢主宰受 200 伤害
- 转嫁 50% = 100 伤害分到 3 队皇家守卫 = 每队 33.3
- 主宰实际承受 100 伤害 + 自身防御 22 / 法术抗性等减免

实战意义：**主宰在皇家守卫阵列存在时几乎不可能被秒杀**。即使敌方陨石雨打 250 伤害到主宰 → 转嫁 125 到 3 队皇家守卫 → 每队 41.7 → 各队损失 1-2 个皇家守卫，主宰只受 125 伤害。

## 抗物理 25% + 集群护盾扩展

皇家守卫现在**对所有物理伤害减免 25%**：

- 170 血 + 抗物理 25% = 实际有效物理 HP **≈ 227**
- 法术伤害不减（陨石雨 / 火球依然满伤）

**集群护盾扩展**：从"同种 + 幼虫 / 工虫"扩展到**所有虫族单位**：

- 一队皇家守卫光环 → 相邻 1 格内所有虫巢单位 -2 物理
- 配合精英战斗工虫的集群护盾被动 → 战斗工虫吃 -3 物理减免（皇家守卫 -2 + 自身被动 -1）
- 这是虫巢"反物理冲锋"的核心阵列协同

## 与 [圣堂骑士](/users/paladin/) / [死亡骑士](/users/death-knight/) 的对比

六级近战兵的耐久比较：

| 单位 | 血量 | 物理减免 | 实际有效物理 HP |
|---|---|---|---|
| [圣堂骑士](/users/paladin/) | 110 | 防御 18 (高) | ≈ 145 |
| [死亡骑士](/users/death-knight/) | 130 | 抗物理 25% | ≈ 173 |
| **皇家守卫** | 170 | 抗物理 25% | **≈ 227** |
| [战独角兽](/users/war-unicorn/) | 115 | 法抗 75% (非物理) | ≈ 138 物理 |

**皇家守卫的物理耐久是六级兵之冠**。但代价是输出最低（20 攻 vs 圣堂骑士 18 × 双重打击）。

## 与维斯帕的协同

[维斯帕](/heroes/vespa/) 的蜂群协同对皇家守卫的集群 buff 上限从 +60% → +90%。

但皇家守卫的核心价值**不在输出**，而在**核心保护**。维斯帕的战术让皇家守卫**从战斗开始就在主宰相邻** —— 触发守护蜂后转嫁。

虫巢的决战标准阵型：

```
  [皇家守卫] [皇家守卫]
[蜂巢主宰 / 远古蜂后]
  [皇家守卫] [战斗工虫 ×3]
```

3 队皇家守卫 + 主宰相邻 + 战斗工虫填空隙 = 主宰受到伤害的 50% 转嫁到 3 队 + 各队再 -25% 抗物理 = **主宰几乎无伤**。

## 使用要点

**站位**：永远紧贴主宰。**单兵突进 = 失去转嫁价值**。

**与精英战斗工虫的分工**：

- 皇家守卫：保护主宰 + 集群护盾覆盖
- 精英战斗工虫：主动输出 + 蜂群冲锋
- 两者**不需要相邻**（不同种），但应该都在主宰光环（集群女王 +2/+2）范围内

**对线神殿**：皇家守卫的抗物理对 [圣堂骑士](/users/paladin/) 的双重打击有效，但**圣战士的"神圣武装"对虫族判定为暗影系？—— 本作规则不明，主流认为虫族不算暗影 / 亡灵，不受神圣武装加成**。皇家守卫面对圣战士是中性对位。

**对线学院**：皇家守卫的抗物理 25% **对法术 0 用**。[陨石雨](/spells/meteor-shower/) 在皇家守卫身上依然是 150-250 伤害 / 格。**虫巢对线学院时皇家守卫保护主宰的能力打折**（陨石雨直接清场 + 转嫁 50% 让皇家守卫自己也阵亡）。

**对线 [亡灵](/factions/necropolis/)**：皇家守卫的物理减免让 [行尸](/users/walking-dead/) / [死亡骑士](/users/death-knight/) 的近战输出大幅缩水。**亡灵打虫巢的核心威胁是凋零律法转化**，皇家守卫的高血量 = 死后大量凋零产物给亡灵。
