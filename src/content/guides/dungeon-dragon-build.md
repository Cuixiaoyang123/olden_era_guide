> 飞龙流是地牢阵营**最经典也最强**的核心构筑。这篇文章不是"看一眼就行"的简介 —— 我们会把每个时间节点的资源、建筑、兵种、英雄技能、律法选择全部讲透。读完之后你可以照着打，不需要再查别的文档。

## 一句话核心

> **飞龙流 = 用前期穴居人 / 幼龙人快速积累 → 第三周末出梦魇 / 暴风之主 → 第五周决战双龙阵。**

整个流派的胜负点在第三周。**第三周末如果龙穴还没解锁，就不是飞龙流了**。

## 英雄：撒迦还是其他？

**首选 [撒迦](/heroes/shakti/)** —— "龙血"专精对所有龙系单位（巨蜥、风暴龙、黑龙）+2 攻防，**正好对应飞龙流的核心阵容**。

撒迦的技能树规划：

1. **5 级**：神秘 → 进阶（每日 +2 法力）
2. **10 级**：智慧 → 进阶（解锁四级法术）
3. **15 级**：领军学 → 基础（最大军队 + 1）
4. **20 级**：抗性 → 基础（法术抗性）
5. **25 级**：智慧 → 高级（解锁五级法术，准备决战）

**不要点**：

- 攻击（撒迦的物理输出不是核心，被龙血专精覆盖）
- 弓术（地牢没有远程主力）
- 鹰眼（被动技能价值低于战术节奏）

## 律法选择路径

飞龙流的标准律法三选：

1. **暗影律法（专属，50 繁荣度）** —— D10 必点
   - 法术伤害的 10% 转化为繁荣度
   - **整个飞龙流的经济发动机**。撒迦的法术伤害 → 繁荣度 → 加速第二条律法兑换

2. **黄金律法（基础，25 繁荣度）** —— D24 左右
   - 每天首都额外 +500 金
   - 第三周末的资源缺口主要在金币，这条律法直接缓解

3. **战时律法（基础，25 繁荣度）** —— D35+，决战前
   - 所有兵种攻击 + 1
   - 每周扣 1500 金 —— 决战前才点，避免长期负担

**为什么不点"防御律法"**：飞龙流的核心是"先动 + 输出爆发"，防御律法对地牢的飞龙阵容收益有限。

## 关键时间节点 (Time Pressure)

| 节点 | 必达指标 | 失败的后果 |
|---|---|---|
| D7 | 幼龙人升级完成 | 第二周清野效率下降 30% |
| D14 | 魔法行会 3 级 + 火球 / 寒冰 | 第三周打不出爆发 |
| D17 | 美杜莎女王升级完成 | 失去反击免疫，远程输出脆 |
| D20 | 风暴龙巢穴建好 | 第四周阵容硬碰硬 |
| D22 | 暴风之主升级完成 | 决战缺核心 |
| D29 | 龙穴解锁 | 黑龙永远不能上场 |

**任何一个节点延迟两天，整条飞龙流就废了。** 资源紧张时必须**砍掉次要建筑**（例如市集、寻宝者小屋）保关键节点。

## 对线决策树

### 对线神殿

**核心威胁**：[圣堂骑士](/units/paladin/) 的双重打击、[大天使](/units/archangel/) 的复活术、[大牧师](/units/high-cleric/) 的治疗。

**对策**：

- **第二周决战**：用 [美杜莎女王](/units/medusa-queen/) 的石化锁住大牧师 → 神殿失去治疗体系崩盘
- **第四周决战**：用 [暴风之主](/units/storm-lord/) 链式闪电对圣堂骑士（金属单位 +50% 伤害）
- **第五周后**：[红龙](/units/red-dragon/) / [黑龙](/units/black-dragon/) 对天使系双倍伤害 —— 一回合带走大天使

**避坑**：不要硬碰圣堂骑士的双重打击。圣堂骑士打梦魇是亏的，**应该绕过去先点大牧师**。

### 对线林地

**核心威胁**：[矮精灵](/units/sprite/) 的 T1 齐射、[凤凰](/units/phoenix/) 的初动 13、半人马的反击免疫。

**对策**：

- **抢初动**：地牢初动均值 9-11，林地均值 10-13。**飞龙流的核心反击是寒冰射线 -2 速度**，把矮精灵 / 凤凰从"先动"打成"后动"
- **绕过半人马**：半人马反击免疫但血量薄。用 [幼龙人](/units/infernal-troglodyte/) 远程慢慢消耗，不要让梦魇硬冲
- **凤凰的复活反制**：凤凰复活 50% 数量后**血量是满的**。打凤凰必须一波带走 + 黑龙补刀复活后的残队

### 对线亡灵

**核心威胁**：骷髅滚雪球、亡灵法术抗性。

**对策**：

- **法术抗性反制**：亡灵对地牢的法术伤害有 +30% 抗性。**飞龙流对线亡灵时减少法术依赖，靠龙系硬碰**
- **骷髅清理**：[美杜莎女王](/units/medusa-queen/) 的远程是对骷髅最高效的清理。**反击免疫让美杜莎对骷髅 0 损耗换血**

### 对线虫巢

**核心威胁**：集群协同 buff。

**对策**：

- **AOE 破阵**：[火球](/spells/fireball/) 是虫巢密集阵的天敌。**飞龙流对虫巢时优先升级火系魔法**
- **梦魇恐惧光环**：梦魇的 -1 / -2 攻击 debuff 直接削弱虫巢集群 buff。这是地牢对虫巢的天然优势

## 常见 Trap

### Trap 1：第一周就建魔法行会三级

**错误原因**：第一周魔法行会 3 级需要 6000 金，相当于放弃幼龙人升级 + 哈比窝建造。**第一周的法术只够用魔法箭**，三级法术等到第二周末才有意义。

### Trap 2：放弃美杜莎升级

**错误原因**：美杜莎女王的"反击免疫"是质变。基础 [美杜莎](/units/medusa/) 看起来够用，但被任何近战贴脸 = 直接被反击拖死。

### Trap 3：跳过风暴龙直接冲黑龙

**错误原因**：黑龙巢穴前置严苛（龙穴升级 + 巫师塔 3 级 + 魔法行会 5 级）。**没有暴风之主的过渡，第四周决战会被对方爆发流压制**。

### Trap 4：第二英雄学法术

**错误原因**：地牢副英雄的核心任务是"清野 + 抢宝物"，不参与决战。给副英雄学法术是浪费经验 —— 应该让副英雄主点领军学 / 战术 / 鹰眼。

### Trap 5：囤繁荣度不点律法

**错误原因**：暗影律法 50 繁荣度的兑换条件在 D10-D12 必达。延迟到 D15 + 兑换 = 第二周决战时少 5 天的繁荣度积累 = 第三周资源缺口拉大。

## 一周内"短局"版本

如果对手提前进攻或地图较小，无法走到第五周黑龙决战，**有简化版**：

- D1-D7：标准飞龙流第一周
- D8-D14：跳过美杜莎，直接抢梦魇巢穴
- D15-D21：6 梦魇 + 14 哈比女王 + 9 暗影巨蜥之眼 决战

这套"快攻版"在 Arena 模式（资源固定 / 短地图）中胜率高于完整飞龙流。
