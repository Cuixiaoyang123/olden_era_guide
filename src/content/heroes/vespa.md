虫巢阵营的招牌英雄。**蜂群协同专精**让集群律法的攻击 buff 上限从 +50% 提升到 **+75%（每相邻一只 +7.5% 攻击）**。在密集阵型中是质变属性。

## 起手三周路线

虫巢的开局比亡灵更难 —— 维斯帕的起手三周决定整局生死。**没有任何前期容错**。

**Week 1**
- D1-3：用 30 幼虫 + 8 工虫**集中冲击野怪点**（虫巢不能分兵 —— 集群律法的 buff 依赖密集）
- D4：抢繁荣度，目标 D7 前达到 50（集群律法门槛）
- D5：建刺虫巢穴（三级兵前置）
- D6-7：**D7 必点集群律法**

**Week 2**
- D8-10：建造喷酸虫池（四级兵核心）
- D11-12：解锁四级法术（[迟缓术](/spells/slow/) / [加速术](/spells/haste/)）
- D13-14：维斯帕到达 5 级，**蜂群协同专精开始生效** —— 集群 buff 上限从 +50% → +75%

**Week 3**
- D15：升级喷酸虫
- D16-21：战斗工虫上线，准备四级法术学习
- 第三周末：阵容 = 80 幼虫 + 25 工虫 + 14 刺虫 + 9 喷酸虫 + 5 战斗工虫

## 推荐技能轮

维斯帕每升 5 级可选 1 个技能：

1. **领军学 → 进阶**（最大军队 + 2，每天 +500 移动）—— 核心
2. **战术 → 进阶**（战斗开始前可调整阵型 5 格 —— 对密集阵型至关重要）
3. **抗性 → 基础**（对法术伤害 -10%，对线法师阵营必备）
4. **智慧 → 基础**（解锁三级法术：抗性术 / 防护术 —— 反法师核心）
5. **攻击 → 基础**（虫巢单兵攻击弱，加 1 攻是 5-10% 提升）

**不推荐**：

- 鹰眼（学院 / 神殿的法术对虫巢无意义）
- 神秘（虫巢英雄法力上限低，多法力意义有限）
- 弓术（虫巢只有一个远程兵种 [喷酸虫](/units/spitter/)，弓术性价比低）

## 战术专精详解

战术技能让维斯帕**在战斗开始前可调整己方阵型 N 格**：

| 等级 | 调整距离 |
|---|---|
| 无 | 0 格（默认阵型） |
| 基础 | 3 格 |
| 进阶 | 5 格 |
| 高级 | 7 格 |
| 专家 | 整个我方半场 |

**为什么虫巢必须高战术**：虫巢的集群律法依赖密集阵型，但默认起始阵型经常不够紧凑。战术让维斯帕能**在战斗开始前把所有单位贴在一起**，触发最大 +50% / +75% 攻击 buff。

不点战术的虫巢英雄实质上**集群律法只生效一半**。

## 副英雄推荐

虫巢的副英雄推荐**纯辅助型** —— 给主英雄维斯帕送兵 / 抢宝物。维斯帕是唯一在前线的英雄，副英雄不要参战。
